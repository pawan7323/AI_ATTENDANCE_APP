 SnapClass – AI Attendance Management System

SnapClass is an AI-powered attendance management system developed using **Python**, **Streamlit**, and **Supabase**. It automates classroom attendance by recognizing students through **face recognition** and **voice recognition**, making the attendance process faster, more accurate, and efficient.

##  Features

-  Teacher and Student portals
- 🔐 Secure login and account creation
- 📷 Face-based attendance using AI
- 🎤 Voice-based attendance using AI
- 👤 Student profile creation with face and voice registration
- 📚 Subject creation and management
- 🔗 Student enrollment using subject codes
- 📊 Attendance history and statistics
- ☁️ Cloud database integration using Supabase
- 🎨 Clean and responsive Streamlit user interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Supabase
- OpenCV
- NumPy
- Pandas
- Speech Recognition
- Machine Learning
- HTML & CSS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/AI_ATTENDANCE_APP.git
```

2. Navigate to the project folder

```bash
cd AI_ATTENDANCE_APP
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Configure your Supabase credentials in Streamlit Secrets.

5. Run the application

```bash
streamlit run app.py
```

## 📸 Modules

### Student Portal
- Create student account
- Register face images
- Register voice sample
- Join subjects using subject code
- View attendance records

### Teacher Portal
- Create and manage subjects
- Take attendance using face recognition
- Take attendance using voice recognition
- View attendance reports
- Share subject codes with students

## 🎯 Objective

The objective of SnapClass is to replace manual attendance with an AI-powered solution that improves speed, accuracy, and classroom efficiency while reducing paperwork.
